// `(::lfun(args))` — common in dunemud (`return(::monster_died(ob));`).
// Used to fail because the lexer greedily preferred `(:` (inline closure
// open) over `(` + `::`. Fixed by spelling the inline-closure open as
// `(` + `token.immediate(':')` instead of an atomic `(:` token, so the
// `::` token wins when both `(:` and `(` `::` could lex.
mixed f() {
  return (::reset(1));
}
