For-init mixing typed decl with bare expression
================================================================================

// `comma_expr_decl` element is either an `expr0` or a typed decl, in
// any order. Mixed-mode case ensures the rule isn't restricted to all-
// declarators.
void f(int *arr) {
  for(int i = 0, sizeof(arr); i < 3; i++) {}
}
